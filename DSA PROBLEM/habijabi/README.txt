How to execute junit tests:
-MUST CHANGE THE DATA PATH INSIDE BOTH Assigncontroll.java and Alltest.java 
 depending on your local machine where the sample data is located.
-install eclipse IDE (if u dont have already)
-open the IDE. from file->import->existing project as workspace -> select Assign25Framework folder.
-open Alltest.java inside the IDE. Right click on it ->run as -> junit test. you should see all tests tick marked on the left side of the screen now.
